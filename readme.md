# Wookielight

A [LimeChat](http://limechat.net/mac/) theme. Each user is assigned one of 16 unqiue colors.

## Installation

Just clone the repo and run bootstrap script:

    $ ./bootstrap.sh

This works with both an App Store and download installation of Limechat.
